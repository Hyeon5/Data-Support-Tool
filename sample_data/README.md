# 샘플 데이터 안내

모든 샘플은 **CSV / XLSX 두 형식(동일 데이터)** 으로 제공되어, 두 업로드 방식을 모두 검증하실 수 있습니다.
각 샘플은 해당 기능만 켜고 실행하시면 결과를 가장 명확하게 확인하실 수 있습니다.

## 개별 기능 테스트용 (9종)

| 파일 | 확인할 기능 | 기대 결과 |
|------|------------|-----------|
| `sample_01_결측치` | 결측치 탐지 | 3·7·11·15·18행에서 결측(빈 값/공백/NULL) 탐지 (총 5행) |
| `sample_02_이상치` | 이상치 탐지 (IQR/Z-Score/ML) | 10·20·30·40행의 극단값 탐지 — IQR만 켜면 정확히 4행 |
| `sample_03_중복데이터` | 중복 데이터 탐지 | 6=7행, 15=16=17행 완전 중복 (총 5행) |
| `sample_04_타입추론` | 데이터 타입 자동 추론 | 정수·실수→Number, 날짜 2형식→Date, 불리언·문자열·숫자문자혼합→String |
| `sample_05_품질점수` | 데이터 품질 종합 점검 | 결측 3건 + 중복 1쌍 + 나이 250/구매금액 950만 이상치 동시 탐지 |
| `sample_06_기술통계` | 기술통계(최소/최대 등) | 숫자 컬럼 설정 표에서 국어/수학/영어의 실제 최소·최대 확인 |
| `sample_07_상관관계` | 상관관계(다변량) | 광고비↑→매출↑(양의 상관), 반품비용↓(음의 상관) — Isolation Forest 로 복합 이상 확인 |
| `sample_08_분포분석` | 분포별 탐지 차이 | 치우친 분포에서 IQR 과 Z-Score 의 탐지 개수가 서로 다름 |
| `sample_09_파라미터테스트` | 알고리즘 파라미터 반영 | Z-Score 2.5→3행 / 3.0→2행, 증감률 50%→3건 / 100%→2건 / 200%→1건 |
