# 데이터 이상치 탐지 · 정제 도구 (설치 불필요, 단일 HTML)

**`support-tool.html` 파일 하나**로 동작하는 데이터 품질 도구입니다. 인터넷·설치·서버 없이,
**웹 브라우저로 열기만 하면** 바로 사용할 수 있습니다. (현재 버전: **V1.1**)

- 📂 지원 형식: **XLSX · XLS · CSV**
- 🔒 **모든 처리는 내 컴퓨터(브라우저) 안에서만** 이루어집니다 — 데이터가 외부로 전송되지 않습니다.
- 🧩 파이썬·Node·엑셀 등 **아무것도 설치하지 않아도** 됩니다.

## 무엇을 할 수 있나요?

| 모드 | 하는 일 |
|------|---------|
| **단일 파일 분석** | 결측치·중복·이상치를 자동으로 찾아 표시하고 Excel로 저장 |
| **비교 검증** | 두 파일(예: 작년 ↔ 올해)을 비교해 추가·삭제·변경·이상 변동을 정리 |
| **개인정보 비식별** | 이름·전화·주민번호 등을 자동 인식해 마스킹·삭제·가명 처리 |
| **정제·형식 변환** | 공공데이터포털 등록용 데이터 정제 + XLSX↔CSV 변환(여러 파일 한 번에) |

---

## 빠른 시작 (3단계)

1. **`index.html` 을 더블클릭**해서 브라우저로 엽니다. (권장: 최신 Chrome / Edge)
2. 화면 **왼쪽 메뉴**에서 원하는 작업을 고릅니다.
   (좁은 화면·모바일에서는 상단 탭으로 표시됩니다.)
3. 파일을 올리고 버튼을 누르면 끝입니다. 결과는 버튼으로 내려받습니다.

> **화면 테마**: 좌측 사이드바 맨 아래의 **☀️ 라이트 / 🌙 다크** 버튼으로 밝은/어두운 테마를
> 전환할 수 있습니다. 선택한 테마는 브라우저에 기억되어 다음에 열 때도 유지됩니다.

---

## ① 단일 파일 분석

> **표 자동 인식**: 제목·소제목·단위·표 밖(아래·옆) 안내 문구는 자동으로 제외하고 **표만** 인식합니다.
> 컬럼명이 병합된 2행 헤더(대분류+소분류)는 `대분류_소분류` 로 합칩니다.
> 한 시트에 표가 2개 이상이면 각각을 `{시트명}_표1`, `{시트명}_표2` 로 나눠 시트 선택에서 고를 수 있습니다.

1. **파일 올리기** — [파일 선택] 버튼 또는 화면으로 **끌어다 놓기**. csv / xlsx / xls 지원.
2. **분석 옵션 고르기** — 결측치·중복·범위·IQR·Z-Score 는 기본으로 켜져 있습니다.
   증감률·머신러닝(Isolation Forest/LOF/DBSCAN)은 필요할 때 체크하세요.
   *옵션 이름에 마우스를 올리면 설명이 나옵니다.*
   - **분석 대상 컬럼**: 체크한 컬럼만 분석합니다(기본 전체). 해제한 컬럼은 모든 검사에서 제외되며,
     미리보기·결과 다운로드에는 원본 전체 컬럼이 그대로 유지됩니다.
3. **[분석 시작]** — 진행 중에도 화면이 멈추지 않습니다. 결과 요약과 표가 나타납니다.
   - 요약의 **막대그래프**로 사유별 건수를, **분포 그래프**로 이상치 위치를 봅니다(마우스 휠로 확대/축소).
   - 표에서 각 이상 행을 **확인/오탐/수정 필요**로 검토하고 메모를 남길 수 있습니다.
4. **[결과 다운로드]** — 원본에 `이상치 여부·사유·근거·검토 상태·메모` 컬럼이 추가된 Excel.
   - **원본이 xlsx 면 글꼴·색·열 너비·행 높이 등 서식을 그대로 유지**하고 분석 컬럼만 덧붙입니다.

**찾아내는 항목**: 결측치 · 중복 · 범위 초과 · 증감률 · IQR · Z-Score · Isolation Forest · LOF · DBSCAN,
그리고 숫자 칸에 섞인 문자 값(형식 오류)을 항상 함께 표시합니다.
(LOF·DBSCAN 은 2만 행이 넘으면 자동으로 건너뛰고 안내합니다.)

## ② 비교 검증 (두 파일 비교)

1. **파일 A(이전)** 와 **파일 B(이후)** 를 각각 올립니다. (CSV·XLSX 섞어도 됩니다.)
2. **기준 컬럼**(예: 관리번호)을 자동으로 추천합니다 — 필요하면 직접 바꿀 수 있습니다.
3. **[비교 시작]** 을 누르면 행을 짝지어 다음처럼 분류합니다.

| 분류 | 뜻 |
|------|-----|
| 추가 / 삭제 | 한쪽에만 있는 행 |
| 변경 | 값이 달라진 행 (셀별 `이전 → 이후 (+50%)` 강조) |
| 이상 변동 | 급격한 증가/감소 등 유난히 큰 변화 **및 결측값 변경** |
| 타입 변경 | 숫자↔문자, 날짜↔문자 전환 |

요약과 필터를 제공하며 **[비교 결과 다운로드]** 로 색상이 적용된 Excel(Compare/Summary)을 받습니다.

## ③ 개인정보 비식별

1. **파일 올리기** — 개인정보가 든 표(csv/xlsx/xls).
2. **자동 유형 인식** — 컬럼별로 주민등록번호·휴대전화·이메일·카드번호·계좌번호·주소·성명·공장등록번호 등을 추정합니다.
   *산단명·기업명 등 회사/시설 이름은 개인정보가 아니므로 제외되고, 등록일 같은 일반 날짜는 오인식하지 않습니다.*
3. **처리 방식 선택**(컬럼별) — 후 [적용] 체크.

| 방식 | 예시 |
|------|------|
| 마스킹 | `홍길동 → 홍**`, `010-1234-5678 → 010-****-5678` |
| 삭제 | 값 비우기 |
| 가명 | 같은 값은 항상 같은 `익명1`, `익명2` 로 치환 |
| 일반화 | 생년월일 → `1990년대`, 주소 → 시·도/시군구 수준, 숫자 → 구간 범주화(`3,450` → `3,000 ~ 3,999`) |
| 해시 | 되돌릴 수 없는 값으로 변환(솔트 입력 필요) |

4. **[비식별화 하기]** 로 상위 20행 미리보기를 확인하고, **다운로드**(비식별결과 + 처리요약 2시트).

## ④ 데이터 정제 · 형식 변환

공공데이터포털 등록 등을 위해 **데이터를 깨끗하게 정리하고 형식을 바꾸는** 기능입니다.
**여러 파일을 한 번에** 처리합니다.

**1) 변환 방식 고르기**

| 방식 | 설명 |
|------|------|
| XLSX → XLSX | 정제 후 **원본 서식 그대로** 저장 |
| XLSX → CSV | 시트별 CSV 로 저장 |
| CSV → CSV | 정제 후 CSV 로 저장 |
| CSV → XLSX | CSV 를 엑셀로 변환 |

> **표 자동 인식**: XLSX→CSV·CSV→CSV·CSV→XLSX 변환은 표만 추려 출력합니다 — 제목·단위·표 밖 텍스트는
> 제외되고, 한 시트에 표가 2개 이상이면 `_표1`, `_표2` 파일/시트로 나눠 저장합니다.
> (XLSX→XLSX 는 원본 서식 보존이 목적이므로 시트 구조를 그대로 유지합니다)

**2) 파일 올리기** — 여러 개를 끌어다 놓거나 선택(최대 30개 · 총 300MB).

**3) 정제 옵션** — 각 항목 옆 **(i) 아이콘**에 마우스를 올리면 설명이 나옵니다.
- 기본 정제(권장, 켜져 있음): **앞뒤 공백 제거 · 연속 공백 정리 · Tab 문자 제거**
- 선택 옵션(필요 시 체크):
  줄바꿈 제거 · **숫자/문자 형식 변환**(숫자 형식 텍스트→숫자, 코드값 보호) · 보이지 않는 문자 제거 ·
  **지수(E)표기 복원**(`1.11E+14` → `111000000000000`, 자릿수 무손실) · **천 단위 구분기호(,) 제거**(`1,234,567` → `1234567`, 소수점 유지) ·
  **모든 특수문자 제거**(유지: 한글·영문·숫자·공백·`.`) ·
  **`"-"`/`"x"`/`"X"` → 0** · **NULL → 0**(XLSX→XLSX 서식 보존에서도 빈 셀을 0 으로 채움) · **NULL → 빈 문자열** ·
  **`"-"`/`"x"`/`"X"` → NULL** · **0 → NULL** · **빈 문자열 → NULL**

**4) [변환 시작]** — **실시간 진행률**(전체 %·현재 파일·시트·처리한 행 수)이 표시되고,
파일별 성공/실패가 함께 나타납니다. 대용량 파일도 화면이 멈추지 않고 처리됩니다.
결과 표에는 **변환 후 파일명**이 나오고, 결과가 여러 개면 **ZIP 으로 한 번에** 내려받습니다.

> **한글 깨짐 방지**: CSV 저장은 기본 **UTF-8 (BOM)** — 윈도우 엑셀에서도 한글이 정상 표시됩니다.
> 불러오기는 UTF-8 / BOM / CP949(EUC-KR) 를 자동 판별합니다(수동 지정 가능).

## 안심하고 쓰셔도 됩니다 (개인정보·보안)

- **외부 전송 없음** — 파일을 서버로 보내지 않고 브라우저 안에서만 처리합니다.
- 화면에 표시되는 값은 자동으로 안전 처리(XSS 방어)되고, 확장자를 위조한 파일은 걸러냅니다.
- 내려받는 Excel/CSV 에서 `=`, `@` 로 시작하는 값은 수식으로 실행되지 않도록 무력화됩니다(수식 주입 방어).

### 참고 사항
- LOF·DBSCAN 은 2만 행 초과 시 자동으로 건너뜁니다(안내 표시).
- 파일 크기: 분석·비식별은 파일당 최대 50MB, 정제·형식 변환은 여러 파일 합쳐 최대 300MB(최대 30개).

<summary>개발자용 정보 (구현·테스트)</summary>

외부 라이브러리 없이 순수 JavaScript 로 구현되어 있습니다. 주요 구성:

- xlsx 읽기: ZIP + 순수 JS DEFLATE 압축 해제 + **시트 XML 문자열 스캔 고속 파서**(예외 시 DOM 폴백, 50k행 기준 약 9배) / xls: OLE2(CFB) + BIFF8 파서
- csv: 인코딩 자동 판별(UTF-8 BOM→UTF-8→CP949), 청크 파싱 / 쓰기: RFC4180 + UTF-8 BOM
- Excel 서식 보존: 원본 워크북을 기반으로 필요한 셀만 수정(XlsxAppender/XlsxCleaner)
- 정제(XLSX→XLSX): DOM 대신 **시트 XML 문자열 스캔·치환**으로 처리(대용량에서 수십 배 빠름),
  행 단위 청크로 UI 양보 + 진행률 보고, 출력은 `CompressionStream` 으로 DEFLATE 압축
- 모든 다운로드는 공통 모듈(Downloader)을 경유하여 동일 로직을 사용
- 분석: 인라인 Web Worker(불가 시 메인 스레드 폴백), 알고리즘 전부 vanilla JS
- 개인정보 비식별: 컬럼명 사전 + 값 정규식 매칭, 순수 JS SHA-256

**파일 구성**

```
index.html                  # 프로그램 본체 (이 파일 하나면 실행)
sample_data/                # 예제 파일 + 안내
scripts/generate_samples.py # 예제 생성 스크립트(실행에는 불필요)
```
