# AI 기반 데이터 이상치 탐지 프로그램 (단일 HTML)

내부망(오프라인) 환경을 위한 **설치가 전혀 필요 없는** 데이터 이상치 탐지 도구입니다.
**`index.html` 파일 하나**로 동작하며, 웹 브라우저로 열기만 하면 실행됩니다.

- **설치 불필요**: 파이썬·Node·npm·pip 아무것도 필요 없습니다.
- **외부 라이브러리 미사용**: CDN·인터넷 통신 없이 순수 JavaScript 로만 구현.
- **오프라인 100%**: 모든 처리가 브라우저 내부에서 이루어지며 데이터가 외부로 전송되지 않습니다.

> XLSX, XLS, CSV — 결측치 · 중복 · 범위 초과 · 증감률 · IQR · Z-Score · Isolation Forest · LOF · DBSCAN

---

## 실행 방법

1. `index.html` 파일을 더블클릭합니다. (또는 브라우저로 파일 열기)
2. 끝입니다. 별도 설치나 서버가 필요 없습니다.

권장 브라우저: 최신 **Chrome / Edge / Firefox**.

---

## 사용 순서

1. **① 파일 선택** — xlsx / xls / csv 파일을 선택합니다. 읽는 즉시 숫자 컬럼 설정 표가 채워집니다.
2. **② 분석 옵션** — 사용할 알고리즘을 체크합니다(기본 전체 선택).
3. **알고리즘 파라미터** — 증감률 임계치, Z-Score 임계치, contamination, n_neighbors, eps, min_samples 조절.
4. **숫자 컬럼 설정** — 컬럼별 허용 최소/최대값을 수정하고 '사용'을 체크합니다.
5. **⑤ 결과 저장 (파일명)** — 저장할 Excel 파일명을 입력합니다.
6. **③ 분석 시작** — 진행률(④)이 표시되며, 완료되면 결과 미리보기가 나타납니다.
7. **결과 다운로드 (Excel)** — 스타일이 적용된 xlsx 파일을 내려받습니다.

## 탐지 알고리즘

| 구분 | 알고리즘 | 설명 |
|------|----------|------|
| 규칙 기반 | 결측치 탐지 | NULL / NaN / 빈 문자열 / 공백 검사 |
| 규칙 기반 | 중복 데이터 | 모든 컬럼 값이 동일한 행 탐지 |
| 규칙 기반 | 범위 검사 | 사용자가 지정한 허용 최소/최대 초과 탐지 |
| 규칙 기반 | 증감률 검사 | 이전 행 대비 증감률(%) 임계치 초과 탐지 |
| 통계 기반 | IQR | Q1-1.5·IQR ~ Q3+1.5·IQR 범위 초과 |
| 통계 기반 | Z-Score | \|z\| > 임계치(기본 3.0) |
| 머신러닝 | Isolation Forest | contamination 조절 (순수 JS 구현) |
| 머신러닝 | Local Outlier Factor (LOF) | n_neighbors 조절 (순수 JS 구현) |
| 머신러닝 | DBSCAN | eps / min_samples 조절, 노이즈 탐지 (순수 JS 구현) |

- **컬럼 타입 자동 분석**: 숫자 / 문자열 / 날짜 자동 판별 (사용자 지정 불필요)
- **복수 알고리즘 동시 판정**: 한 행이 여러 알고리즘에 걸리면 사유를 쉼표로 연결

## 결과 Excel 구조

원본 데이터를 그대로 유지하고 마지막에 컬럼 2개를 추가합니다.

- **이상치 여부**: 이상이면 `이상`, 정상이면 공란
- **이상치 사유**: 예) `범위 초과 [수행률] (허용:0~100, 실제:130), IQR 이상 [수행률]`

| 시트 | 내용 |
|------|------|
| `Result` | 원본 + 이상치 여부/사유. **이상 행 전체를 연한 노란색(#FFF2CC)** 으로 표시 |
| `Anomaly` | 이상으로 판정된 행만 추출 |

## 기술적 참고 (외부 라이브러리 없이 구현한 부분)

| 기능 | 구현 방식 |
|------|-----------|
| xlsx 읽기 | ZIP 파싱 + **순수 JS DEFLATE 압축 해제(tiny-inflate 포팅)** + XML 파싱(DOMParser) |
| xls 읽기 | BIFF8 레코드 파서 (best-effort) |
| csv 읽기 | 따옴표/개행 처리 + UTF-8/EUC-KR 인코딩 자동 감지(TextDecoder) |
| Excel 쓰기 | ZIP(STORED) 직접 생성 + CRC32 + styles.xml(노란색 채우기) |
| 알고리즘 | IQR/Z-Score 및 Isolation Forest/LOF/DBSCAN 전부 vanilla JS |

> `.xls`(구형 이진 형식)는 best-effort 로 지원합니다. 복잡한 서식의 파일은
> `xlsx` 또는 `csv` 로 변환 후 사용하시길 권장합니다.

## 파일 구성

```
index.html          # 프로그램 본체 (이 파일 하나면 실행 가능)
sample_data/        # 테스트용 샘플 데이터
  ├── sample.csv
  └── sample.xlsx
README.md
```

`sample_data/sample.csv` 또는 `sample.xlsx` 를 업로드하면 의도적으로 삽입된
결측치·중복·범위 초과·증감률·통계/ML 이상치를 확인할 수 있습니다.
