# AI 기반 데이터 이상치 탐지 프로그램 (단일 HTML)

내부망(오프라인) 환경을 위한 **설치가 전혀 필요 없는** 데이터 이상치 탐지·비교 검증 도구입니다.
**`index.html` 파일 하나**로 동작하며, 웹 브라우저로 열기만 하면 실행됩니다.

- **설치 불필요**: 파이썬·Node·npm·pip 아무것도 필요 없습니다.
- **외부 라이브러리 미사용**: CDN·인터넷 통신 없이 순수 JavaScript 로만 구현.
- **오프라인 100%**: 모든 처리가 브라우저 내부에서 이루어지며 데이터가 외부로 전송되지 않습니다.

> XLSX, XLS, CSV — 결측치 · 중복 · 범위 초과 · 증감률 · IQR · Z-Score · Isolation Forest · LOF · DBSCAN · 두 파일 비교 검증

---

## 실행 방법

1. `index.html` 파일을 더블클릭합니다. (또는 브라우저로 파일 열기)
2. 끝입니다. 별도 설치나 서버가 필요 없습니다.

권장 브라우저: 최신 **Chrome / Edge / Firefox**.

---

## ① 단일 파일 분석

1. **파일 선택** — 버튼 또는 **Drag & Drop** 으로 csv / xlsx / xls 파일을 올립니다.
   업로드가 끝나면 업로드 영역에 **파일명이 표시**되고, [✕ 파일 제거] 로 초기화할 수 있습니다.
2. **분석 옵션 / 파라미터** — 옵션명에 마우스를 올리면(모바일은 터치) 의미·권장값·주의사항을 담은 **도움말**이 표시됩니다.
3. **숫자 컬럼 설정** — 자동 추출된 숫자 컬럼의 허용 최소/최대를 지정합니다.
4. **분석 시작** — 분석은 **Web Worker(백그라운드)** 에서 실행되어 대용량에서도 화면이 멈추지 않습니다.
5. **결과 다운로드** — 원본 + `이상치 여부`/`이상치 사유` 컬럼, 이상 행 노란색(#FFF2CC),
   `Result`/`Anomaly` 2시트. 날짜 값은 실제 날짜 셀로 저장됩니다.

### 탐지 항목

| 구분 | 검사 | 비고 |
|------|------|------|
| 항상 수행 | **형식 오류** | 숫자 컬럼에 섞인 문자 값(예: `"abc"`)을 별도 사유로 표시 |
| 규칙 | 결측치 / 중복 / 범위 / 증감률 | 증감률 임계치(%) 조절 |
| 통계 | IQR(배수 1.5) / Z-Score(임계치 조절) | |
| 머신러닝 | Isolation Forest / LOF / DBSCAN | 순수 JS 구현. **LOF·DBSCAN 은 2만 행 초과 시 자동 건너뜀 후 안내** |

수행되지 못한 검사가 있으면 결과 화면에 **건너뜀 사유가 표시**됩니다.

## ② 두 파일 비교 검증

상단 **[두 파일 비교 검증]** 탭에서 사용합니다.

1. **파일 A(기준·이전)** 와 **파일 B(비교·이후)** 를 올립니다. CSV↔XLSX 조합도 가능합니다.
2. **기준 컬럼 자동 추론** — 이름 유사도(동의어: 산단명↔산업단지명, 업체명↔회사명 등) +
   고유값 비율 + 값 겹침을 종합해 가장 알맞은 컬럼 쌍을 추천합니다. 직접 변경도 가능합니다.
3. **이상 변동 기준** — 변화율(%) / Z-Score / IQR 를 선택합니다.
   Z-Score·IQR 은 매칭된 모든 행의 변화량(변화 없음=0 포함)을 모수로 계산합니다.
4. **비교 시작** — 결과가 아래와 같이 분류됩니다.

| 분류 | 설명 |
|------|------|
| 추가 / 삭제 | B에만 있거나 A에만 있는 행 (행 내용 스냅샷 표시) |
| 변경 | 값이 달라진 행 — 셀별 `이전 → 이후 (+50%)` 강조 |
| 이상 변동 | 급격한 증가/감소, 통계적으로 유난히 큰 변화 |
| 결측값 변경 | 값↔결측 전환 |
| 타입 변경 | 숫자↔문자, 날짜↔문자 전환 |

요약(총 행/동일/변경/추가/삭제/이상 변동/결측값 변경/타입 변경)과 필터를 제공하며,
**[비교 결과 다운로드]** 로 상태별 색상이 적용된 Excel(Compare/Summary 2시트)을 받을 수 있습니다.

## 기술적 참고 (외부 라이브러리 없이 구현한 부분)

| 기능 | 구현 방식 |
|------|-----------|
| xlsx 읽기 | ZIP 파싱 + **순수 JS DEFLATE 압축 해제(tiny-inflate 포팅)** + XML 파싱(DOMParser) |
| xls 읽기 | **OLE2(CFB) 컨테이너 해석** + BIFF8 레코드 파서, 셀 서식(XF/FORMAT/1904체계) 기반 날짜 변환 |
| csv 읽기 | 따옴표/개행 처리 + UTF-8/EUC-KR 인코딩 자동 감지, 청크 파싱(진행률 표시) |
| Excel 쓰기 | ZIP(STORED) 직접 생성 + CRC32, 노랑/초록/빨강 채움, 날짜 셀(numFmt) |
| 분석 실행 | 인라인 **Web Worker**(불가 환경은 메인 스레드 폴백) |
| 알고리즘 | IQR/Z-Score 및 Isolation Forest/LOF(메모리 O(n·k))/DBSCAN 전부 vanilla JS |

### 보안
- 컬럼명/셀 값은 화면 표시 전 HTML 이스케이프 처리(XSS 방어).
- 확장자 위조는 매직바이트로 차단, 압축 해제 크기 상한(zip-bomb 방어).
- 내려받는 Excel 에서 `=`, `@` 로 시작하는 문자열은 **작은따옴표(') 접두로 무력화**되어
  수식으로 실행되지 않습니다(CSV Injection 방어).

### 알려진 한계
- LOF·DBSCAN 은 2만 행 초과 데이터에서 자동으로 건너뜁니다(결과 화면에 안내).
- 파일 업로드 최대 50MB.
